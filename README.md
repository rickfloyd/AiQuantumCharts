## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone <repository-url>
cd quantum-charts

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production
```bash
npm run build
```
